# Quick Start Guide

## Installation & Run

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start development server:**
   ```bash
   npm run dev
   ```

3. **Open your browser:**
   Navigate to `http://localhost:3000`

## What You'll See

### Home Page
- **Matrix-style code rain** background animation
- **Glitch text effects** on the main title
- **Create Room** button to generate unique room IDs
- **Join Room** input to enter existing room IDs

### Chat Room Page
- **Tab-based navigation** between Chat and Video Call
- **Real-time chat interface** with anonymous messaging
- **Video call interface** with controls
- **Hacking-themed terminal** aesthetics throughout

## Next Steps

After installing dependencies, the UI will be fully functional. To add real-time functionality:

1. Set up a Socket.io server for chat
2. Implement WebRTC signaling for video calls
3. Connect the frontend to your backend
